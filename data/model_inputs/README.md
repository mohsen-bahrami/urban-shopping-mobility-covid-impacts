# Model input tables

The four model input tables (`table_2018.csv` through `table_2021.csv`) are derived from preprocessed mobility, POI, and census features. 

They are not stored in GitHub because each file exceeds GitHub’s file-size limits.

[Click here for urban-shopping-mobility-covid-impacts OSF Repository](https://osf.io/r9yz7/overview)

Download them from the companion OSF repository and place them in this folder before running the full calibration pipeline.
